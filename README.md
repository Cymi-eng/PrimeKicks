
# PrimeKicks
Primekicks is a modern ecommerce website for stylish and trendy shoes.This project is designed to provide users with a smooth online shopping experience for sneakers,casual shoes and sports footwear.

# Project overview
Primekicks has combined clean design,user friendly navigation and modern web tecnologies to create professional online shoe store.The website is built with responsiveness in mind ,ensuring it works perfectly across desktops,tablets and mobile devices.

# Features
* Modern and responsive design
* Interactive homepage hero section
* Featured shoe collections
* Product cards with pricing
* Mobile-friendly layout
* Smooth user experience

 Product Card Components
Each product card includes:

* Product image
* Product title
* Price display
* Rating stars
* Add to Cart button
* About Section
* Contact Section

# Technologies used 
* HTML5
* CSS
* css grid
